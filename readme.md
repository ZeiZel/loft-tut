## SCSS

Была использована конвертация в CSS с помощью Live Sass Compiler с данными настройками внутри VS Code:

```JSON
"liveSassCompile.settings.showOutputWindowOn": "None",
"liveSassCompile.settings.formats": [{
    "format": "expanded",
    "extensionName": ".css",
    "savePath": "/css/"
}]
```
